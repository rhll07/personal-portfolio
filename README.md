# 🌐 Personal Portfolio Website

This is a **static multi-page personal portfolio website** created as part of the **User Interface Design (UID)** lab.

🔗 Live: https://rhll07.github.io/personal-portfolio/

---

## 🧾 About the Project

The website presents my personal, academic, and extracurricular details across multiple pages:

* Resume
* Personal Information
* Travel Blog
* Hobbies

---

## 🛠️ Implementation

* Built using **HTML only**
* **No JavaScript** used
* **No external CSS files**
* Minimal **inline styling** used only for basic alignment
* Uses **Google Fonts** for typography

---

## 🎯 Features

* Multi-page navigation
* Structured content using tables and lists
* Use of semantic HTML elements (`<section>`, `<article>`, etc.)
* Multimedia integration:

  * Images
  * Audio
  * Video
* Hosted using **GitHub Pages**
